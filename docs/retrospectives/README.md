# Retrospectives

This directory contains milestone retrospectives and workflow learning after a
milestone is complete.

Use retrospectives to capture:

- what shipped
- what worked
- what was too heavy
- workflow or process adjustments
- follow-up ideas that should not silently change product scope

Current retrospectives:

- [M1: Local CLI Replay Preview](m1-local-cli-replay-preview.md)
- [M2: Local Executable Distribution](m2-local-executable-distribution.md)
- [M3: Configurable Replay Inputs](m3-configurable-replay-inputs.md)

Retrospectives are historical documents. Active planning for M4 belongs in
[`../milestones/`](../milestones/README.md), not here.
