# Business Analysis

This directory contains concise business and product analysis documents used to
shape roadmap direction before milestone scope is accepted.

Business analysis documents should stay lighter than PRDs, ADRs, and milestone
plans. Use them to classify candidate work, compare dependencies and value, and
recommend next artifacts or candidate milestones.
