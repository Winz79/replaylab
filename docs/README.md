# ReplayLab Documentation

This directory mixes durable product documentation with a small set of
historical planning artifacts from earlier milestones.

Use these folders first:

- [`adr/`](adr/README.md): accepted architecture decisions.
- [`brd/`](brd/README.md): business and product framing for public scope.
- [`prd/`](prd/README.md): product requirements documents for accepted slices.
- [`milestones/`](milestones/README.md): milestone-level planning and scope.
- [`retrospectives/`](retrospectives/README.md): milestone close-out notes and
  workflow learning.

Root-level documents are legacy project-context material from the initial setup
phase. They remain in place for history and link stability, but new milestone
planning should live under `docs/milestones/` and completed-milestone reviews
should live under `docs/retrospectives/`.

- M2 complete
- M3 complete
- M4 is the active planning milestone
- M5 and M6 remain directional only
