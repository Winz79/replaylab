# Milestone Planning

This directory contains milestone-level planning documents.

Use milestone documents to define the scope, slices, and risks for the current
or upcoming milestone without turning the roadmap into a detailed plan.

Current usage:

- completed milestone plans remain here as historical planning artifacts
- completed milestone retrospectives belong in
  [`../retrospectives/`](../retrospectives/README.md)
- detailed product requirements belong in [`../prd/`](../prd/README.md)

Milestone docs currently in this folder:

- [M2: Local Executable Distribution](m2-local-executable-distribution.md)
- [M4: HTTP Sender Preview](m4-http-sender-preview.md)
- [M5: Minimal Web UI](m5-minimal-web-ui.md)
- [M6: Private Adapter Extension Model](m6-private-adapter-extension-model.md)
- [M7: Hostable Entry Points](m7-hostable-entry-points.md)

Milestone planning should add or update documents here rather than expanding
`docs/roadmap.md` into a detailed plan.
