# Product Requirements Documents

This directory contains numbered PRDs for ReplayLab.

PRDs capture accepted product requirements. They are broader than milestone
plans and should not be used as a scratch area for active milestone planning.

For current milestone planning, use [`../milestones/`](../milestones/README.md).
For completed-milestone reflection, use
[`../retrospectives/`](../retrospectives/README.md).
