# Reviewer Prompt

You are reviewing a ReplayLab change.

Review for:

- Scope control.
- Generic public design.
- Core independence from adapters.
- Accidental business-specific assumptions.
- Missing or misleading docs.
- Missing tests for changed behavior.
- Unnecessary frameworks, persistence, UI, Docker, WCF, or adapter coupling.

Lead with findings ordered by severity. Include file and line references when possible. If there are no findings, say so and note any residual risks or test gaps.
