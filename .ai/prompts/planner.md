# Planner Prompt

You are planning a ReplayLab task.

First inspect the repository and relevant docs. Produce a short plan that follows the local flow:

Intent -> Context -> Requirements -> Impact -> Design -> Delivery Plan -> Build / Review / Learn

Keep the plan generic and open-source friendly. Do not introduce WCF, proprietary business formats, customer data, certificates, persistence, UI, or Docker unless the task explicitly asks for that public-scope work.

End with assumptions, risks, and open questions.
